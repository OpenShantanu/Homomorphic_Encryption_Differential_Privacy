To implement HomoEncryption.py, it needs TenSEAL package of Python.
To install TenSEAL package, use pip command.
    
    pip install tenseal

You might face issue with newer version of Python. So you may get Python 3.11 version downloaded.

    winget install Python.Python.3.11

Then install tenseal again.
  
    py -3.11 -m pip install tenseal
